## Dataset

### Source & license

- Synthetic stream generated by `data/make_synthetic_labeled_stream.py` (MIT / your repo’s license)
- Real-data option: **WISDM Smartphone and Smartwatch Activity and Biometrics Dataset (UCI)**, licensed **CC BY 4.0**.
  - Dataset format: `<subject-id, activity-code, timestamp, x, y, z>` per sensor stream.
  - Citation: Weiss, G. (2019). WISDM Smartphone and Smartwatch Activity and Biometrics Dataset [Dataset]. UCI ML Repository. DOI: 10.24432/C5HK59.

### What’s in `data/demo_labeled_stream.csv`

- One row per timepoint (sensor sample)
- Columns:
  - `timestamp`: integer index or time
  - `x,y,z`: sensor values (3-axis)
  - `label`: per-timepoint activity label (string)

## Method

### Windowing and model

- Window size selection tool: `select_window_size_from_csv`
- Evaluation tool: `evaluate_windowed_classifier_from_csv`
- Classifier: `minirocket` (falls back to a simple feature model if aeon isn’t installed)
- Metric optimized: `balanced_accuracy`
- Balance strategy: `segment_cap` (`max_windows_per_segment=25`)
- Train/test split: group-aware split by segment (`test_size=0.25`, `seed=1337`)

### Commands

```bash
ts-agents demo window-classification --no-llm
```
